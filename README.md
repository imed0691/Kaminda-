# Guide d'utilisation - AnkiPro Français

## Introduction

AnkiPro Français est une application web qui vous permet de traduire des mots anglais vers le français et de les mémoriser efficacement grâce à un système de répétition espacée inspiré d'Anki. Cette application fonctionne entièrement dans votre navigateur et stocke vos données localement, sans nécessiter de connexion internet permanente après le chargement initial.

## Fonctionnalités principales

1. **Traduction de mots** : Traduisez des mots ou phrases de l'anglais vers le français
2. **Import de listes** : Importez et traduisez des listes entières de mots
3. **Système de mémorisation** : Utilisez des flashcards avec répétition espacée pour mémoriser efficacement
4. **Tableau de bord** : Suivez votre progression et gérez votre liste de mots
5. **Stockage local** : Toutes vos données sont stockées localement dans votre navigateur

## Comment utiliser l'application

### Traduction de mots

1. Cliquez sur l'onglet "Traduire" dans le menu principal
2. Entrez un mot ou une phrase en anglais dans le champ de saisie
3. Cliquez sur le bouton "Traduire"
4. La traduction apparaîtra en dessous
5. Vous pouvez ajouter ce mot à votre liste d'apprentissage en cliquant sur "Ajouter à ma liste"
6. Vous pouvez également écouter la prononciation du mot en cliquant sur l'icône du haut-parleur

### Import de listes

1. Cliquez sur l'onglet "Importer" dans le menu principal
2. Vous avez deux options :
   - Coller une liste de mots (un mot par ligne) dans la zone de texte
   - Importer un fichier texte contenant une liste de mots (un mot par ligne)
3. Cliquez sur "Traduire la liste"
4. Une fois la traduction terminée, vous pouvez :
   - Ajouter tous les mots à votre liste d'apprentissage
   - Modifier individuellement les traductions si nécessaire
   - Supprimer certains mots de la liste

### Apprentissage avec flashcards

1. Cliquez sur l'onglet "Apprendre" dans le menu principal
2. Une carte apparaîtra avec un mot en anglais
3. Essayez de vous rappeler la traduction française
4. Cliquez sur la carte pour la retourner et voir la traduction
5. Évaluez votre connaissance du mot en cliquant sur l'un des boutons :
   - "Difficile" : Le mot reviendra rapidement dans votre session
   - "Correct" : Le mot reviendra dans un délai moyen
   - "Facile" : Le mot reviendra après un délai plus long
6. Continuez jusqu'à ce que vous ayez terminé toutes les cartes pour la journée

### Tableau de bord

1. Cliquez sur l'onglet "Tableau de bord" dans le menu principal
2. Vous y verrez tous les mots de votre liste d'apprentissage
3. Vous pouvez :
   - Rechercher des mots spécifiques
   - Filtrer les mots par statut (nouveaux, en apprentissage, à réviser, maîtrisés)
   - Modifier ou supprimer des mots
   - Exporter vos données pour les sauvegarder
   - Importer des données précédemment exportées
   - Réinitialiser toutes vos données (attention, cette action est irréversible)

### Paramètres

1. Cliquez sur "Paramètres" en bas de page
2. Vous pouvez personnaliser :
   - Le nombre de cartes par jour
   - Le nombre de nouvelles cartes par jour
   - Le thème de l'application (clair, sombre, automatique)

## Conseils d'utilisation

- **Régularité** : Pour une mémorisation efficace, utilisez l'application quotidiennement
- **Petites sessions** : Il est préférable de faire plusieurs petites sessions d'apprentissage plutôt qu'une seule longue session
- **Sauvegardez vos données** : Utilisez régulièrement la fonction d'exportation pour sauvegarder vos données
- **Personnalisez les paramètres** : Ajustez le nombre de cartes quotidiennes selon votre disponibilité

## Fonctionnement hors ligne

Une fois l'application chargée, la plupart des fonctionnalités fonctionnent hors ligne. Cependant, la traduction de nouveaux mots nécessite une connexion internet.

## Compatibilité

AnkiPro Français est compatible avec tous les navigateurs modernes :
- Google Chrome
- Mozilla Firefox
- Microsoft Edge
- Safari
- Opera

L'application est également responsive et s'adapte aux écrans de toutes tailles (ordinateurs, tablettes, smartphones).
