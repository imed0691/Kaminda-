À propos

AnkiPro Français est une application web inspirée d'Anki, conçue pour vous aider à apprendre l'anglais efficacement grâce à un système de répétition espacée.

Cette application utilise le stockage local de votre navigateur pour sauvegarder vos mots et votre progression.

Fonctionnalités

  • Traduction anglais-français
  • Import de listes de mots
  • Système de flashcards avec répétition espacée
  • Suivi de progression
